# 股票盯盘应用 - FinTech AI 前端

这是一个基于React和TypeScript的股票盯盘应用前端项目，用于展示和监控股票价格。

## 功能特点

- 股票列表展示
- 股票价格监控
- 价格上下限预警
- 股票搜索

## 项目结构

```
src/
  ├── api/        # API服务
  ├── assets/     # 静态资源
  ├── components/ # 可复用组件
  ├── pages/      # 页面组件
  └── ...
```

## 可用脚本

在项目目录中，可以运行:

### `npm start`

在开发模式下运行应用
打开 [http://localhost:3000](http://localhost:3000) 在浏览器中查看

### `npm test`

启动测试运行器

### `npm run build`

构建生产版本应用到 `build` 文件夹

## 技术栈

- React
- TypeScript
- CSS3

## 开发指南

1. 克隆仓库
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm start`
